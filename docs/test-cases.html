Contoh Implementasi Test Case

check_circle TC-001 - Positive Case: Risk Input Berhasil Disimpan
Test IDTC-ERM-RI-001
ScenarioRisk berhasil dibuat dan masuk ke tab Need Follow Up
ModuleRisk Input - BAU
Pre-conditionUser sudah login sebagai Risk Owner dengan reference aktif. Current control evaluation tersedia.
Test Steps
  1. Klik menu Risk Input.
  2. Isi semua field wajib: Risk Description, Inherent Risk (Likelihood + Severity), Current Control Evaluation = "Require to Improve", Action = "Reduce", Residual Risk Plan, Objective, BSC.
  3. Tambahkan 1 mitigasi dengan PIC valid, Due Date, dan Periode.
  4. Klik Save.
Test DataRisk Owner: dewichatarina, PIC: rini.hervina, Likelihood: Possible, Severity: Major, Action: Reduce
Expected ResultRisk tersimpan dengan Risk Number format RN-BU-2026-XXX. Data muncul di tab Need Follow Up. Risk Owner menerima email notifikasi.
StatusPASSED
cancel TC-002 - Negative Case: Validasi Mandatory Mitigasi
Test IDTC-ERM-RI-002
ScenarioSistem menolak penyimpanan risk yang wajib mitigasi tetapi field mitigasi dikosongkan
ModuleRisk Input - BAU
Pre-conditionUser login sebagai Risk Owner. Current control evaluation tersedia.
Test Steps
  1. Buka menu Risk Input.
  2. Isi semua field kecuali mitigasi.
  3. Set Action = "Reduce" (bukan Adequate/Avoid).
  4. Klik Save tanpa menambahkan mitigasi.
Test DataAction: Reduce, Mitigasi: kosong
Expected ResultSistem menampilkan validasi error. Risk tidak tersimpan.
StatusPASSED
cancel TC-003 - Negative Case: Update Cancelled Mitigation
Test IDTC-ERM-MIT-001
ScenarioMitigasi dengan status Cancelled tidak bisa diedit oleh PIC
ModuleMitigation Update Required - PIC
Pre-conditionTerdapat risk dengan mitigasi yang sudah berstatus Cancelled. User login sebagai PIC mitigasi tersebut.
Test Steps
  1. Login sebagai PIC (contoh: rini.hervina).
  2. Buka menu Mitigation Update Required.
  3. Cari risk dengan mitigasi Cancelled (ref: RN-BU-2026-611).
  4. Coba edit field mitigasi yang Cancelled tersebut.
Test DataRisk Number: RN-BU-2026-611, Status mitigasi: Cancelled
Expected ResultSeluruh kolom mitigasi yang Cancelled ter-block/disabled. PIC tidak bisa melakukan perubahan apapun.
StatusPASSED
warning TC-004 - Edge Case: Prorate Plan Mitigasi
Test IDTC-ERM-MIT-002
ScenarioPerhitungan prorate actual mitigation progress tidak lagi menghitung mitigasi yang sudah di-cancel
ModuleRisk Register - Registered / Mitigation Progress
Pre-conditionTerdapat risk BAU dengan lebih dari satu mitigasi aktif. Salah satu mitigasi sudah di-cancel oleh Risk Owner. PIC mitigasi lain sudah melakukan update progress.
Test Steps
  1. Login sebagai Risk Owner.
  2. Cancel salah satu mitigasi pada risk RN-BU-2026-623.
  3. Login sebagai PIC mitigasi yang masih aktif.
  4. Update progress mitigasi aktif tersebut.
  5. Cek kolom Actual Cumulative Average di RR dan Dashboard L1/L2.
Test DataRisk: RN-BU-2026-623, Jumlah mitigasi: 2, Status: 1 aktif - 1 cancelled
Expected ResultPerhitungan prorate actual hanya menghitung mitigasi aktif. Mitigasi Cancelled tidak ikut sebagai pembagi. Nilai average di RR dan Dashboard konsisten.
StatusPASSED
